# Spec Change Triggers — Rust

Rust-specific examples for the [trigger](../triggers.md) categories.

## Toolchain Changes

- Rust edition update (e.g., edition 2024).
- New clippy lint that conflicts with existing patterns.
- Change in `rustfmt` defaults.

## Key Dependency Evolution

Major version changes in core ecosystem crates:

- `tokio` — async runtime
- `tracing` — instrumentation
- `thiserror` — error derive
- `serde` — serialization
